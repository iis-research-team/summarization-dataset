Для эффективного освоения залежей углеводородов чрезвычайно важной становится высокая точность проводки горизонтальных скважин в целевых пластах-коллекторах. В процессе бурения геонавигация скважины со сложной траекторией выполняется по геофизическим данным в реальном масштабе времени. Представленная работа посвящена разработке нового программного обеспечения для геонавигации нефтегазовых скважин с горизонтальным завершением. Для решения задач геонавигации используются алгоритмы, основанные на методах корреляции каротажных данных и численной инверсии измерений электромагнитного каротажа.
Разработанное новое программное приложение основано на веб-технологиях и имеет клиент-серверную архитектуру. Для оптимизации времени выполнения ресурсоемких расчетов используются высокопроизводительные облачные вычисления.
